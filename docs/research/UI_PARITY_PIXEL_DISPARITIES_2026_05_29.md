- **Evidence:** MAIN_MENU_TITLE_TEXT_RENDER_GHIDRA_REPORT.md §6 confidence table ('Color is #FFFF00 yellow — HIGH content, MEDIUM RGB-byte-order') and §8 open question (yellow vs cyan)
- **Verifier:** Rust side confirmed: `src/app_main_menu_shell_render.rs:22` defines `SHELL_BUTTON_TEXT_RGB_FFFF00 = [1.0, 1.0, 0.0]` (R=1.0, G=1.0, B=0.0 = yellow #FFFF00).
